Introduction

Review management systems allow organizations to conduct periodic reviews over task allocation to employees and provide ratings that help track work efficiency. Performance review reports support decisions about which departments or employees are performing better and where training is required.

How to use this module

Use the Manager Dashboard to add, edit, delete and analyse performance reviews for employees. Employees can use their dashboard view to see only their own reviews.
